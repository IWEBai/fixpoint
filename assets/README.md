# Fixpoint Assets

Branding assets for GitHub App and Marketplace.

## Required for Phase 1

### Logo

- **Size:** 128×128 minimum (GitHub App); 256×256+ for Marketplace
- **Format:** PNG, square
- **File:** `logo.png`
- **Current placeholder:** `fixpoint-logo.png` (verify usage rights before publishing)
- **Usage:** GitHub App settings, landing page, README

### Feature Card (Marketplace)

- **Size:** 1280×640 pixels
- **Format:** PNG or JPG
- **File:** `feature-card.png`
- **Usage:** GitHub Marketplace listing (Phase 2)

## Placeholder

Until final assets are ready, use a simple icon or text-based placeholder. GitHub accepts any square image for the app logo. Verify license/attribution for any temporary asset before release.
