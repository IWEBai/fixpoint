Privacy Policy

Last updated: February 2025

1. Data We Collect

When you install Fixpoint as a GitHub App, we receive webhook payloads from GitHub that may include:

  • Installation ID and account information (organization or user name)
  • Repository names and metadata
  • Pull request content (diffs, file paths, code) for repositories where Fixpoint is installed

2. How We Use It

We use this data solely to:

  • Run security scans on changed files in pull requests
  • Apply fixes or propose fixes when vulnerabilities are found
  • Post comments and status checks on pull requests

3. Data We Do Not Sell

We do not sell, rent, or share your data with third parties for marketing or advertising.

4. GitHub Terms of Service

Fixpoint operates under GitHub's Terms of Service. Your use of the app is also subject to GitHub's policies.

5. Contact

Questions? Contact us at support@fixpoint.dev.
